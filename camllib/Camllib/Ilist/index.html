val flatten : ?depth:int -> ('a, 'b) t -> ('a, 'b) t

Flatten the recursive list, only starting from the given

  • flatten [] = []
  • flatten [a;[b;[c];d];e;[f]] = [a;b;c;d;e;f]
  • flatten ~depth:2 [a;[b;[c];d];e;[f]] = [a;[b;c;d];e;[f]]
  • flatten ~depth:3 [a;[b;[c];d];e;[f]] = [a;[b;[c];d];e;[f]]
