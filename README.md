Camllib
=======

About
-----
Various abstract datatypes inspired by the standard OCaml library.

To build
--------

```
dune build
```

To install
----------

```
opam install .
```

If you just want to install the library without modifications:

```
opam pin add git+https://github.com/jogiet/camllib.git#master
```
